## 🚀 Features

- 🎙️ Record audio using device microphone
- 📂 List saved audio recordings
- ▶️ Play recordings with playback listener
- 🗑️ Delete individual recordings

---

## 🛠️ Tech Stack

- **React Native**
- **TypeScript**
- **Redux Toolkit**
- **react-native-audio-recorder-player**
- **react-native-fs**
